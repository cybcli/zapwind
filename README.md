
# Zapwind

A simple text editor built using Python.

## Shortcuts

- `Ctrl + O` - Open file
- `Ctrl + S` - Save current file
- `Ctrl + N` - Create new file
- `Ctrl + T` - Switch theme
- `Ctrl + Q` - Save as current file
- `Ctrl + V` - Paste content
